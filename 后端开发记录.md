#### 初始化项目


```bash

```


#### 项目目录

```bash
|-- message-server
    |-- index.js			// 启动项目
    |-- package.json		 
    |-- yarn.lock
    |-- 后端开发记录.md		// 项目笔记
    |-- .vscode
    |   |-- settings.json
    |-- config
    |   |-- default.js		// 项目配置
    |-- controller			// 路由等处理逻辑
    |-- db					// 数据库
    |-- middleware			// 中间件
    |-- public				// 静态资源
    |-- routers				// 路由
    |-- utils				// 工具
```
